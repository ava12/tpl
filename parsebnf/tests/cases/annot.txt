Ожидаемый результат теста:

# @result

либо

# @result JSON

либо:

# @result:
# многострочный
# JSON

Ожидаемые ошибки:

# @error char
  некорректный символ
# @error defined нетерминал
  нетерминал уже определен
# @error empty
  отсутствуют правила
# @error eof
  неожиданный конец файла
# @error no-definition нетерминал
  не определен тетерминал
# @error no-terminal нетерминал+
  невозможно разрешить зависимости
# @error quote
  отсутствует закрывающая кавычка
# @error token лексема ожидаемая_лексема*
  неожиданная лексема
# @error undecidable нетерминал терминал
  неоднозначная грамматика
